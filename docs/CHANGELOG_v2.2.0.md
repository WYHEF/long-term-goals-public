## 🎉 重大更新概览

本次 v2.2.0 版本是一个重大更新，主要包含**项目结构重组**和**邮件提醒功能完善**两大核心改进。

---

## ✨ 新增功能

### 📧 邮件提醒系统完善
- **自动邮件提醒**: 每日定时发送目标提醒邮件
- **多时段提醒**: 支持北京时间 8:00、20:00、22:00 三个时段
- **个性化内容**: 包含今日到期倒数日和每日任务清单
- **美观模板**: 精心设计的 HTML 邮件模板
- **状态监控**: 邮件发送状态监控和失败重试机制

### 🔄 定时任务优化
- **GitHub Actions 优化**: 提前1小时执行，减少延迟影响
- **备用方案**: 支持 cron-job.org 等外部定时服务
- **双重保障**: 主备系统确保邮件发送可靠性
