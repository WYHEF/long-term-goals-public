# 名言警句功能说明

## 功能概述

用户可以自定义管理自己的名言警句/座右铭库，每次访问"今日任务"页面时，系统会随机展示一条激励语，帮助用户保持动力和专注。

## 功能特点

✅ **个人定制**：每个用户拥有独立的名言库  
✅ **随机展示**：每次刷新页面自动切换名言  
✅ **手动切换**：点击刷新按钮立即换一句  
✅ **灵活管理**：可添加、编辑、删除、启用/禁用名言  
✅ **预置内容**：新用户自动获得6条经典名言（可删除）  
✅ **风格统一**：界面设计与整体系统保持一致

## 数据库结构

### quotes 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | UUID | 主键 |
| user_id | UUID | 用户ID（外键） |
| content | TEXT | 名言内容 |
| author | VARCHAR(100) | 作者/来源（可选） |
| is_active | BOOLEAN | 是否启用 |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

**索引**：
- `idx_quotes_user_id`：按用户查询
- `idx_quotes_is_active`：按启用状态过滤

**RLS策略**：启用行级安全，用户只能访问自己的数据

## 使用方法

### 1. 查看名言（今日任务页面）

1. 访问首页 `/` 或 "今日任务" 页面
2. 页面顶部会显示一条随机名言
3. 点击 🔄 按钮可以切换到另一条名言

**展示规则**：
- 只展示已启用的名言
- 随机选择，避免连续重复
- 如果没有启用的名言，显示提示引导用户添加

### 2. 管理名言（设置页面）

访问 `/settings` 设置页面，找到 "💬 名言警句管理" 卡片。

#### 添加名言
1. 点击 "+ 添加新名言" 按钮
2. 输入名言内容（必填）
3. 输入作者/来源（可选）
4. 点击 "✅ 添加" 保存

#### 编辑名言
1. 找到要编辑的名言
2. 点击 ✏️ 编辑按钮
3. 修改内容和作者
4. 点击 "保存" 或 "取消"

#### 启用/禁用名言
- 点击名言右侧的开关按钮
- 绿色表示已启用，灰色表示已禁用
- 禁用的名言不会在首页显示

#### 删除名言
1. 点击 🗑️ 删除按钮
2. 确认删除操作
3. 预置的名言也可以删除

## 技术实现

### 前端组件
- **QuoteCard.vue**：名言展示卡片组件
- **SettingsView.vue**：名言管理界面

### 状态管理
- **quotes.js (Pinia Store)**：管理名言数据和操作

### 主要API
- `fetchQuotes()`：获取用户的所有名言
- `getRandomQuote()`：随机获取一条启用的名言
- `addQuote(content, author)`：添加新名言
- `updateQuote(id, updates)`：更新名言
- `deleteQuote(id)`：删除名言
- `toggleActive(id)`：切换启用状态

## 部署说明

   - 检查 Supabase Dashboard 中的 quotes 表
   - 确认已启用 Row Level Security
   - 验证四个策略都已创建

3. **无需额外配置**
   - 前端代码已集成
   - Store已注册
   - 组件已引入

## 注意事项

⚠️ **数据隔离**：每个用户的名言库完全独立，互不影响

⚠️ **删除限制**：如果删除所有名言或禁用所有名言，首页会显示提示，不影响其他功能

⚠️ **性能考虑**：名言列表会在设置页面和首页分别加载，使用Pinia缓存避免重复请求
