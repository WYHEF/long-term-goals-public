# 倒数日功能说明

## 功能概述

倒数日功能允许用户添加重要日期并实时查看距离该日期还有多少天，适用于考试、生日、纪念日、截止日期等场景。系统会在首页显示紧急倒数日提醒卡片。

## 主要特性

### 1. **倒数日管理**
- ✅ 添加倒数日（事件名称、日期时间、描述）
- ✅ 编辑倒数日
- ✅ 删除倒数日
- ✅ 标记完成（已过期的倒数日）
- ✅ 置顶显示（重要倒数日）

### 2. **分类系统**
- 📝 **考试** - 重要考试提醒
- 🎂 **生日** - 亲友生日记录
- 💝 **纪念日** - 重要纪念日
- ⏰ **截止日期** - 项目/任务截止
- 📌 **其他** - 自定义事件

### 3. **颜色标记**
- 蓝色、红色、绿色、黄色、紫色、粉色
- 用于区分不同类型或重要程度的倒数日

### 4. **智能提醒**
- **首页显示**：30天内或置顶的倒数日自动显示在今日任务页面顶部
- **倒计时颜色**：
  - ≤ 3天：红色（紧急）
  - 4-7天：橙色（临近）
  - > 7天：蓝色（正常）
- **详细倒计时**：30天内的事件显示精确的"X天X时X分"

### 5. **筛选功能**
- 全部倒数日
- 即将到来（未过期）
- 已过期

## 数据库结构

### countdowns 表

```sql
CREATE TABLE countdowns (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES auth.users(id),
  title VARCHAR(255) NOT NULL,           -- 事件名称
  description TEXT,                      -- 描述
  target_date TIMESTAMP WITH TIME ZONE,  -- 目标日期时间
  category VARCHAR(50),                  -- 分类
  color VARCHAR(20),                     -- 颜色标记
  remind_days_before INTEGER[],          -- 提前提醒天数
  is_completed BOOLEAN,                  -- 是否完成
  hide_after_passed BOOLEAN,             -- 过期后隐藏
  is_pinned BOOLEAN,                     -- 是否置顶
  created_at TIMESTAMP,
  updated_at TIMESTAMP
);
```

## 用户体验

### 1. 首页显示规则
- 显示条件：
  - 距离目标日期 ≤ 30天
  - 或 已置顶
  - 且未标记为完成
- 显示数量：最多3个
- 排序：置顶 > 时间最近

### 2. 倒计时显示
- **未过期**：
  - 大号数字显示剩余天数
  - 30天内显示详细倒计时（天时分）
  - 根据剩余时间显示不同颜色
- **已过期**：
  - 灰色显示"已过期"
  - 显示过期天数
  - 可标记为"已完成"后隐藏

### 3. 卡片样式
- 左侧边框颜色由用户选择
- 分类图标显示
- 置顶倒数日有黄色边框高亮
- Hover 时有阴影效果

## API 接口

### 获取倒数日列表
```javascript
await countdownsStore.fetchCountdowns()
```

### 创建倒数日
```javascript
await countdownsStore.createCountdown({
  user_id: userId,
  title: '英语四级考试',
  description: '加油！',
  target_date: '2025-06-15T09:00:00+08:00',
  category: 'exam',
  color: 'red',
  is_pinned: true
})
```

### 编辑倒数日
```javascript
await countdownsStore.updateCountdown(id, {
  title: '新标题',
  is_pinned: false
})
```

### 删除倒数日
```javascript
await countdownsStore.deleteCountdown(id)
```

### 置顶/取消置顶
```javascript
await countdownsStore.togglePin(id)
```

### 标记完成
```javascript
await countdownsStore.markAsCompleted(id, true)
```

const diffMs = targetDate.diff(now)

## 常见问题

### Q1: 为什么首页没有显示倒数日？
A: 检查以下条件：
1. 是否距离目标日期 ≤ 30天
2. 是否置顶
3. 是否标记为完成
4. 最多只显示3个

### Q2: 如何让某个倒数日一直显示在首页？
A: 点击"置顶"按钮，置顶的倒数日会一直显示，不受30天限制

### Q3: 过期的倒数日会自动删除吗？
A: 不会自动删除，需要手动标记为"已完成"或删除

### Q4: 时间显示不准确怎么办？
A: 确保浏览器时区设置正确，系统会自动根据本地时区显示
