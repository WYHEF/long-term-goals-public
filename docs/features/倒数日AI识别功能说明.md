# 倒数日AI智能识别功能说明

## 功能概述

在添加倒数日时，用户可以直接粘贴包含日期时间信息的通知文字（如会议通知、考试通知等），系统使用DeepSeek AI自动识别并提取关键信息，自动填充到表单中。

## 功能特点

✅ **智能识别**：自动提取标题、日期、时间、分类  
✅ **一键填充**：识别成功后自动填充所有字段  
✅ **置信度提示**：显示AI识别的准确度（高/中/低）  
✅ **错误提示**：识别失败时给出明确提示  
✅ **保留原文**：描述框内容不会被清除  

## 使用方法

### 步骤1：粘贴文字

在"添加倒数日"弹窗中，将包含日期时间的通知文字粘贴到**描述**框中。

**示例文字：**
```
【通知】应学院党总支要求，各支部需召开二级党组织师生党员研讨交流会。
本科生第四党支部研讨交流会安排如下：
会议时间：12月18日晚20:00
会议地点：390-563-038
研讨主题：坚定自信、挺膺担当，以青春实践书写文化新篇
```

### 步骤2：点击AI识别

点击描述框下方的 **"✨ AI自动识别"** 按钮。

### 步骤3：等待识别

系统会显示"⏳ AI识别中..."，通常需要2-5秒。

### 步骤4：查看结果

- **识别成功**：表单自动填充，显示绿色提示"✅ 识别成功！"
- **识别失败**：显示红色错误提示，需要手动填写

### 步骤5：确认并保存

检查自动填充的信息是否正确，如有需要可以手动调整，然后点击"添加"保存。

## 识别能力

### 支持识别的内容

| 字段 | 识别能力 | 示例 |
|------|----------|------|
| **事件标题** | 自动总结关键信息（10字以内） | "党支部研讨交流会" |
| **日期** | 识别各种日期格式 | "12月18日" → 2025-12-18 |
| **时间** | 识别12/24小时制 | "晚20:00" → 20:00 |
| **分类** | 自动判断事件类型 | 会议 → "其他" |

### 日期格式支持

- ✅ "12月18日"
- ✅ "2025年12月18日"
- ✅ "2025-12-18"
- ✅ "明天"、"后天"（相对日期）
- ✅ "下周一"、"本月底"

### 时间格式支持

- ✅ "20:00"、"20点"
- ✅ "晚上8点"、"上午9点"
- ✅ "8:00 PM"
- ⚠️ 如果没有时间，默认为 00:00

### 分类识别规则

| 分类 | 关键词 | 图标 |
|------|--------|------|
| **考试** | 考试、测试、考核、答辩、笔试、面试 | 📝 |
| **生日** | 生日、诞辰、寿辰 | 🎂 |
| **纪念日** | 纪念日、周年、结婚纪念 | 💝 |
| **截止日** | 提交、上交、报名截止、申请截止、deadline | ⏰ |
| **其他** | 会议、活动、聚会、讲座等 | 📌 |

## 技术实现

### 核心函数

**`parseCountdownInfo(description)`** - 位于 `src/config/deepseek.js`

```javascript
// 调用示例
const result = await parseCountdownInfo(descriptionText)

// 返回格式
{
  success: true,
  data: {
    title: "事件标题",
    date: "2025-12-18",
    time: "20:00",
    category: "other",
    confidence: "high"
  }
}
```

### AI提示词设计

- 明确指定输出格式（JSON）
- 提供当前日期作为参考
- 包含分类规则说明
- 设置合理的默认值
- 降低temperature（0.3）提高准确性

### 错误处理

1. **AI功能未启用**：提示管理员未开启AI功能
2. **网络错误**：显示"AI识别失败"
3. **JSON解析失败**：返回"无法识别文字中的关键信息"
4. **置信度低**：仍然填充，但提示用户检查

## 注意事项

⚠️ **AI识别不是100%准确**
- 建议在保存前检查自动填充的信息
- 特别注意日期和时间是否正确
- 如果识别错误，可以手动修改

⚠️ **需要DeepSeek API配置**
- 确保管理员已在系统设置中启用AI功能
- 确保API Key配置正确

⚠️ **网络延迟**
- AI识别通常需要2-5秒
- 识别过程中按钮会禁用，避免重复提交

⚠️ **文字格式要求**
- 文字中应包含明确的日期和时间信息
- 如果信息模糊，可能识别不准确

## 配置要求

### 1. DeepSeek API配置

在 `.env` 文件中配置：
```env
VITE_DEEPSEEK_API_URL=你的API地址
VITE_DEEPSEEK_API_KEY=你的API密钥
```

### 2. 系统设置

管理员在"系统设置"中启用"DeepSeek AI功能"。
