# 更新日志 - 倒数日AI智能识别功能

## 版本 v2.4.0 - 2025-12-18

### 🎉 新增功能

#### ✨ 倒数日AI智能识别

**功能描述**：
用户添加倒数日时，可以直接粘贴包含日期时间的通知文字，AI自动识别并提取关键信息，一键填充表单。

**核心特性**：
- ✅ 智能提取事件标题（自动总结）
- ✅ 识别各种日期格式（相对日期、绝对日期）
- ✅ 识别时间（12/24小时制）
- ✅ 自动分类（考试/生日/纪念日/截止日/其他）
- ✅ 置信度评估（高/中/低）
- ✅ 一键填充所有字段

### 📊 AI识别能力

#### 支持的日期格式
```
✅ "12月18日" → 2025-12-18
✅ "2025年12月18日" → 2025-12-18
✅ "2025-12-18" → 2025-12-18
✅ "明天" → 自动计算
✅ "下周一" → 自动计算
```

#### 支持的时间格式
```
✅ "20:00" → 20:00
✅ "晚上8点" → 20:00
✅ "上午9点" → 09:00
✅ "8:00 PM" → 20:00
⚠️ 无时间 → 00:00（默认）
```

#### 智能分类规则
| 分类 | 触发关键词 | 图标 |
|------|-----------|------|
| 考试 | 考试、测试、考核、答辩 | 📝 |
| 生日 | 生日、诞辰 | 🎂 |
| 纪念日 | 纪念日、周年 | 💝 |
| 截止日 | 提交、截止、deadline | ⏰ |
| 其他 | 会议、活动、聚会 | 📌 |

### 💡 使用场景

#### 场景1：考试通知
```
输入：英语四级考试将于2025年12月15日上午9:00举行
输出：
  标题：英语四级考试
  日期：2025-12-15
  时间：09:00
  分类：考试 📝
```

#### 场景2：会议通知
```
输入：【通知】明天下午3点在会议室召开项目讨论会
输出：
  标题：项目讨论会
  日期：2025-12-19（自动计算）
  时间：15:00
  分类：其他 📌
```

#### 场景3：作业截止
```

  时间：23:59
  分类：截止日 ⏰
```

### 🔧 技术细节

#### AI提示词优化
- 明确当前日期作为参考
- 详细的分类规则说明
- 严格的JSON输出格式要求
- 合理的默认值设定
- Temperature=0.3（提高准确性）

#### 错误处理机制
1. **AI功能未启用** → 提示联系管理员
2. **网络请求失败** → 显示具体错误信息
3. **JSON解析失败** → 提示无法识别
4. **必填字段缺失** → 使用默认值

#### 安全性
- 使用现有的DeepSeek API配置
- 遵循系统AI功能开关
- 不存储AI识别历史（隐私保护）
- 用户可随时手动修改AI填充的内容

### ⚙️ 配置要求

#### 环境变量
```env
VITE_DEEPSEEK_API_URL=你的API地址
VITE_DEEPSEEK_API_KEY=你的API密钥
```

#### 系统设置
管理员需在"系统设置"中启用"DeepSeek AI功能"。

### 🚀 部署步骤

1. **无需数据库变更** - 复用现有倒数日表结构
2. **无需额外配置** - 使用现有DeepSeek API
3. **直接部署前端代码** - 重启应用即可

```bash
# 开发环境
npm run dev

# 生产环境
npm run build
```

### 📝 使用说明

1. 点击"添加倒数日"
2. 在描述框粘贴通知文字
3. 点击"✨ AI自动识别"
4. 等待2-5秒
5. 检查自动填充的信息
6. 确认无误后保存

### ⚠️ 注意事项

- AI识别不是100%准确，建议保存前检查
- 需要管理员启用AI功能
- 识别通常需要2-5秒
- 文字中应包含明确的日期时间信息
